웹 이벤트란?

인게임 배너를 통해 접속할 수 있는 웹페이지로 슈퍼스타 시리즈의 메인 컨텐츠인 리듬 게임 외에 유저에게 다른 재미를 제공하기 위해 만들어진 이벤트입니다.

전체적인 흐름은 유저 접속 → 접속한 유저 데이터 조회 → 데이터 기반 로직 실행(ex. 게임 플레이 횟수를 포인트화) → 유저 액션(요청) → 액션 처리(응답) 순입니다.

※ 규정 상 작업물 URL은 제공하지 못하였습니다.
